👁️ Blindful

Accessibility Navigation Extension

⌨️ Keyboard Shortcuts

Navigate elements Tab
Navigate backwards Shift+Tab
Activate element Enter
Toggle voice input Ctrl+Z
Text to Speech Ctrl+Shift+P

🎤 Voice Commands (English)

"go to 5" — jump to element 5
"next" — move to next element
"previous" — move to previous element
"click" — activate focused element
"read page" — describe the page
"scroll down" — scroll the page down
"scroll up" — scroll the page up
"stop" — stop speaking
