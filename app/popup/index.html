Keyboard

Toggle voiceCtrl+Z

Voice Commands

"go to 5" — jump to element 5
"next" / "previous" — navigate elements
"click" — activate focused element
"read page" — describe the page
"scroll down / up" — scroll page
"zoom in / out" — change text size
"reset text" — restore text size
"stop" — stop speaking
